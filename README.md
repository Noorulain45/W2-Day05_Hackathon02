📌 Project Overview

W2-Day05 Hackathon02 is a web development project created as part of a hackathon challenge. The goal of the project is to build a responsive and interactive website using modern front-end technologies.
The project demonstrates how to create a structured website with dynamic features, clean UI design, and smooth user interactions.

🛠️ Technologies Used
This project was developed using the following technologies:
HTML5 – For structuring the web pages
CSS3 / Tailwind CSS – For styling and responsive design
JavaScript – For interactivity and dynamic behavior
JSON – For managing and loading data dynamically

✨ Features
Responsive and modern UI design
Dynamic content rendering using JavaScript
Smooth navigation between section
Organized and reusable code structure
Clean and readable layout for better user experience

🎯 Learning Outcomes
Through this project, the following concepts were practiced:
DOM manipulation with JavaScript
Working with external data using JSON
Creating responsive layouts
